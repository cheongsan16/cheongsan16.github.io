# Matplotlib

다양한 종류의 방대한 데이터의 구조와 패턴을 쉽게 파악하기 위해 그래프를 사용한다. 판다스 내 데이터 시각화를 지원하는 기능이 있지만, 풍부하지 않기 때문에 다른 전문 도구를 활용한다.<br>
그 대표적인 도구가 Matplotlib 라이브러리이다. Matplotlib은 2차원 평면 그래프에 관한 다양한 포맷과 기능을 지원한다.
